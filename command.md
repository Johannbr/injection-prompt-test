# Exemples de Commandes Bash

Voici quatre exemples de commandes bash utiles :

## 1. Lister les fichiers avec détails
```bash
ls -lah
```
Liste tous les fichiers du répertoire courant avec permissions, taille et fichiers cachés.

## 2. Rechercher un texte dans les fichiers
```bash
grep -r "pattern" /chemin/du/repertoire
```
Recherche récursivement un pattern de texte dans tous les fichiers d'un répertoire.

## 3. Afficher l'utilisation du disque
```bash
du -sh *
```
Affiche la taille de chaque élément dans le répertoire courant de manière lisible.

## 4. Trouver des fichiers modifiés récemment
```bash
find . -type f -mtime -7
```
Trouve tous les fichiers modifiés dans les 7 derniers jours.
